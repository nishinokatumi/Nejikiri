■フォルダ、ファイルの説明
-----
■01ゲーム操作方法
　　　　HowToPlay.txt
　　　　NejikiriHowToPlay.pdf

  ・ファイルの内容
　　　　コントローラー・キーボードの操作方法です。
　　　　キーボードでも動きますが、できるのであればコントローラーで遊んでいただきたいです。

■02ゲーム内容説明
　　　　NejikiriPlanning.pdf
　　　　ProjectSummary.txt
　　　　MyAssignment.txt

  ・ファイルの内容
　　　　「NejikiriPlanning.pdf」にゲームの企画
　　　　「ProjectSummary.txt」にプロジェクト概要
　　　　「MyAssignment.txt」に私の担当箇所が記載されています。

■03動画
　　　　NejikiriPV.mp4
　　　　NejikiriPV.wmv
　　　　03readme.txt

  ・ファイルの内容
　　　　ゲームの流れを動画にまとめております。
　　　　念のため2つの拡張子のデータがありますが内容は同じです。
　　　　「.mp4」のほうが高画質に再生されます。

■04実行形式
　　　　resource
　　　　nejikiri.exe
　　　　04readme.txt

  ・ファイルの内容
　　　　実際にゲームを遊ぶ時のファイルになります。
　　　　「.exe」を起動すると始まります。
　　　　「resouce」フォルダはゲーム内で使用されている画像・音などが入っております。
　　　　参照されるものなので、「.exe」ファイルと同じ階層（フォルダ）内においてください。

■05ライブラリ
　　　　プロジェクトに追加すべきファイル_VC用
　　　　05readme.txt

  ・ファイルの内容
　　　　使用ライブラリについてです。
　　　　プログラムをビルドするために必要なライブラリです。
　　　　ビルドする際は、
　　　　C:の直下に「C:\DxLib_VC」
　　　　というパスになるようにコピーしてください。

■06ソースファイル
　　　　nejikiriguru
　　　　nejikiri.sln

  ・ファイルの内容
　　　　プロジェクトのソースです。
　　　　「nejikiriguru」の中に「resource」「source」が入っています。
　　　　「resource」にはゲームで使用する素材、「source」には実際に書いたコードファイルがあります。
　　　　参照しているので同じ階層（フォルダ）内に入れてください。

■07アピールポイント
　　　　AppealPoints.txt

  ・ファイルの内容
　　　　制作時の役割・こだわったところを記述しております。
